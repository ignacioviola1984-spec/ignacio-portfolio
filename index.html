How I workCómo trabajo

Validate firstValidar primero

I start from a real problem with a real user, not from the tech. Phase 0 is always understanding the process and cleaning the data. Empiezo por un problema real con un usuario real, no por la tecnología. La Fase 0 siempre es entender el proceso y limpiar los datos.

Build fastConstruir rápido

I turn validated problems into working, deployed systems in days, building with Claude Code. Convierto problemas validados en sistemas funcionando y desplegados en días, construyendo con Claude Code.

Human in the loopHumano en el loop

Every stage requires a subject-matter-expert approval before the flow moves forward. Nothing ships without sign-off. Cada etapa requiere la aprobación de un experto antes de que el flujo avance. Nada se publica sin firma.

Deterministic by designDeterminista por diseño

Numbers come from deterministic code, never invented. If a figure isn't there, the system errors instead of guessing. LLMs handle the narrative, not the math. Los números salen de código determinista, nunca inventados. Si un dato no está, el sistema da error en vez de adivinar. Los LLM manejan la narrativa, no la matemática.
