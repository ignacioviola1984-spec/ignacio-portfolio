How I work

Validate first

I start from a real problem with a real user, not from the tech. Phase 0 is always understanding the process and cleaning the data.

Build fast

I turn validated problems into working, deployed systems in days, building with Claude Code.

Human in the loop

Every stage requires a subject-matter-expert approval before the flow moves forward. Nothing ships without sign-off.

Deterministic by design

Numbers come from deterministic code, never invented. If a figure isn't there, the system errors instead of guessing. LLMs handle the narrative, not the math.
